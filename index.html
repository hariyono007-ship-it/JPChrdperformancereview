📖 Cara Setup Google Sheets:
1. Buka Google Sheets → buat spreadsheet baru
2. Salin Spreadsheet ID dari URL (antara /d/ dan /edit)
3. Di Google Sheets, klik Extensions → Apps Script
4. Tempel script dari tombol di bawah → Deploy → New Deployment
5. Pilih type: Web App → Execute as: Me → Who has access: Anyone
6. Salin URL hasil deployment ke field di atas
